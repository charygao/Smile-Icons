# Smile Icons

VSCode 图标插件，这是我所喜欢的样式，全部高清重绘

![preview](preview.png)

## 没有你要的图标？

- 可以通过任何方式联系我来设计添加
- 也可以你自己设计好了，把图标给我，需要按照项目中 **icons.sketch** 文件中的示例来设计，最终输出一个 **svg** 格式的文件即可
